# 유럽 두 나라 여행 🇫🇷 🇨🇭

프랑스 · 스위스 2개국 10일 여행 기록 (2026.06.12 – 06.21). 독일(프랑크푸르트)은 출국 경유.
사진 위주의 정적 웹사이트입니다. 빌드 도구 없이 브라우저로 바로 열려요.

## 구성
- **약도** — 방문 도시를 순서대로 이은 경로 약도 (맨 위 한 번)
- **일정** — 출국부터 귀국까지 날짜 순 타임라인 + 그날 사진
- **모아보기** — 전체 사진 갤러리 (나라별 필터 + 크게보기)

## 📷 사진 넣는 법 (제일 중요)
1. 그날 사진을 해당 **날짜 폴더**에 넣습니다.
   예) 6/13 사진 → `images/06-13/`
2. 파일 이름은 **`01.jpg`, `02.jpg`, `03.jpg` …** 순서대로 (두 자리, 빈 번호 없이).
3. `data.js` 에서 그 날의 **`count`** 를 넣은 사진 장수로 바꿉니다.
   - 예: 6/13에 5장 넣었으면 → `count: 5`
   - PNG 를 넣었으면 그 날에 `ext: "png"` 추가

> 💡 휴대폰 사진이 `.HEIC` 면 브라우저에서 안 보입니다. `.jpg` / `.png` 로 변환해서 넣어주세요.

폴더 목록: `images/1일차(6.12)` ~ `images/10일차(6.21)` (이미 만들어져 있음)

### 묶음(그룹)으로 보여주기 — 권장 방식
하루 사진을 "공항 / 기내식 / 에펠탑 / 점심 …" 처럼 **장소·식사 단위로 묶어서** 보여줄 수 있어요.
사진을 넣을 때 파일 이름을 이렇게 지어주세요 (한글 설명 OK):

```
<묶음번호>_<순서> <설명>.jpeg
예) 1_1 공항에서 기다리는중.jpeg
    2_1 기내식 닭고기.jpeg
    6-2 에펠 전망대.jpeg
```
> 같은 묶음번호 안에 식사·장소가 섞여 있으면, 넣어주실 때 "출발/도착은 1일차로" 같이 알려주시면
> 일차/묶음을 나눠서 정리합니다. (1일차 예시처럼 파일명은 `d1-02-1.jpeg` 식으로 자동 정리)
- **묶음번호**가 같으면 한 묶음으로 묶이고, 번호 순서대로 위→아래 정렬됩니다.
- 설명은 그 묶음의 **소제목**으로 사이트에 표시돼요. (식사류는 🍽, 장소는 📍 자동)
- 넣고 나서 알려주시면, 파일명을 깨짐 없는 ASCII(`d1-02-1.jpeg`)로 정리하고 `data.js` 의
  `groups` 에 묶어 넣어드립니다. → `data.js` 의 `groups` 형식 참고 (1일차에 예시 있음).

> 그냥 순서대로만 보여주고 싶으면 묶음 없이 `count`(사진 장수)만 바꿔도 됩니다.

## 🎬 동영상 넣는 법 (웹에서 바로 재생)
1. 같은 날짜 폴더에 동영상을 넣고 이름을 **`v01.mp4`, `v02.mp4` …** 로.
2. `data.js` 에서 그 날의 **`vcount`** 를 동영상 개수로 바꿉니다. (사진 `count` 와 별개)
3. 갤러리/타임라인에서 **재생 버튼을 누르면 다운로드 없이 바로 재생**됩니다. (인라인 재생)

포맷·용량 주의:
- **MP4 (H.264 + AAC)** 권장 — 모든 브라우저에서 인라인 재생됩니다.
- 아이폰 `.mov` 는 크롬/안드로이드에서 안 될 수 있어요 → `.mp4` 로 변환 권장.
- **GitHub 파일 1개당 100MB 제한** (50MB 넘으면 경고). 긴 영상은 압축하거나,
  용량이 크면 유튜브에 올리고 링크/임베드를 쓰는 편이 낫습니다.
